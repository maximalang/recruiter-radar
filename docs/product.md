# Recruiter Radar

## Что это
Сервис, который ищет компании с признаками активного найма и отправляет пользователю готовые лиды в Telegram.

## Для кого
Для рекрутеров, агентств и специалистов по найму.

## Что должен делать MVP
1. Получать вакансии из one primary jobs source.
2. Определять компании с признаками спроса на найм.
3. Считать простой score.
4. Отправлять лиды в Telegram.
5. Давать возможность менять статус лида.

## Phased expansion strategy

### Phase 1, доказать ценность на one primary jobs source
- Стартуем с одного основного источника вакансий, чтобы быстро проверить scoring, delivery и полезность лидов.
- Источник запуска может быть hh.ru или другой jobs source с достаточным покрытием.

### Phase 2, расширить primary hiring-signal sources
После подтверждения MVP добавляем новые источники, которые прямо показывают найм:
- career pages компаний
- LinkedIn jobs и company pages, либо аналогичные внешние jobs/company sources, если нужен более осторожный запуск
- отдельные tech job boards

### Phase 3, добавить enrichment sources
После этого подключаем источники, которые не создают лид сами по себе, но улучшают приоритизацию и контекст:
- ЕГРЮЛ / ФНС для юридических данных компании
- сайт компании для контактов и контекста
- funding signals и другие внешние business signals

## Что не делаем сейчас
- Сложную аналитику
- Большой CRM
- Одновременный запуск многих primary hiring-signal sources в первом релизе
- Красивую сложную админку
